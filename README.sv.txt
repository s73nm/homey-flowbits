FlowBits gör automatisering mer tillgängligt genom att ge dig enkla verktyg för vardagliga situationer. Det fokuserar inte på stora funktioner, utan på de små sakerna som gör skillnad.

Med några extra logikkort blir det lite enklare att bygga ett bra flow.

- Med stegsekvenser går du enkelt igenom olika steg i en bestämd ordning.
- Flaggor används för att snabbt slå funktioner eller tillstånd i flöden av eller på.
- Lägen hjälper dig att tydligt definiera den allmänna situationen i hemmet.
- Inga-upprepningsfönster förhindrar att ett flöde aktiveras flera gånger i snabb följd.
- Timers ger ett enkelt sätt att lägga till fördröjningar eller tidsintervaller i dina automatiseringar.
