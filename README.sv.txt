FlowBits gör automatisering mer tillgänglig genom att ge dig enkla verktyg för vardagliga situationer. Det fokuserar inte på stora funktioner, utan på de små sakerna som gör skillnad.

Med några extra logikkort blir det lite lättare att bygga ett bra flöde.

- Stegsekvenser (cykler) låter dig enkelt gå igenom olika steg i en fast ordning, framåt eller bakåt.
- Flaggar låter dig snabbt slå på eller av funktioner eller tillstånd i dina flöden.
- Lägen hjälper dig att tydligt definiera den allmänna situationen i ditt hem.
- Set gör att du kan gruppera relaterade tillstånd och spåra dem som en helhet. Du kan aktivera tillstånd individuellt, exklusivt eller alla på en gång, med valfri utgångstid.
- Etiketter lagrar textvärden som du kan ställa in, kontrollera och använda i flödena.
- Händelser håller reda på när något hände, vilket gör att du kan kontrollera om det inträffade idag, inom ett visst tidsintervall eller hur många gånger.
- Ingen-upprepning-fönster förhindrar att ett flöde utlöses flera gånger på kort tid.
- Timers ger ett enkelt sätt att inkludera förseningar eller tidsintervall i dina automatiseringar, med stöd för paus och återupptagning.
- Reglage tillhandahåller numeriska värden som kan justeras via flöden eller widgets.
- Signaler erbjuder ett enkelt sätt att skicka och ta emot namngivna signaler mellan flöden.

Ytterligare verktyg:

- Matematiska operationer för avrundning, division, multiplikation och justering av siffror.
- Slumpbaserade villkor för att lägga till slumpmässighet i dina flöden.

Widgets finns tillgängliga för flaggar, lägen, set, etiketter, händelser, timers och reglage.
