FlowBits делает автоматизацию более доступной, предоставляя простые инструменты для повседневных ситуаций. Он не пытается делать что-то глобальное — только маленькие вещи, которые действительно помогают.

С несколькими дополнительными логическими карточками создавать хороший поток становится немного проще.
