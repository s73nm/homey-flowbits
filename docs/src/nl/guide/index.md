# Handleiding

FlowBits is een hulpprogramma voor Homey waarmee je je flows kunt structureren, beheren en uitbreiden zonder nieuwe apparaattypen toe te voegen. Alle functies worden direct vanuit flows gedefinieerd en beheerd, wat je volledige flexibiliteit geeft terwijl je setup overzichtelijk en voorspelbaar blijft.

## Installatie

Installeer FlowBits vanuit de [Homey App Store](https://homey.app/nl-nl/app/com.basmilius.flowbits/FlowBits/) op je Homey. De app vereist geen aanvullende configuratie—zodra deze is geïnstalleerd, zijn alle FlowBits-functies beschikbaar als flow-kaarten.

## Experimentele versie

Er is ook een experimentele versie beschikbaar voor gebruikers die aankomende functies willen testen. Deze versie kan worden gedownload vanaf de [pagina met experimentele versies](https://homey.app/nl-nl/app/com.basmilius.flowbits/FlowBits/test/).
